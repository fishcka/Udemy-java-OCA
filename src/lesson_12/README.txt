Урок 12. Конструкции «if» и «if else». Ternary оператор (ст 151-156)
if else, equals in if, ternary operator

В классе StudentTest написать 2 метода, которые принимают 2 input параметра – 2 объекта класса Student из Lesson11.
Первый метод сравнивает 2-х студентов, используя один if statement и логические операторы внутри него и выводит на
экран информацию о том, равны ли студенты. Второй метод использует nested if statement, сравнивает все
атрибуты студента по отдельности, выводит на экран информацию о том, равны ли студенты, а если не равны,
то в чём именно было обнаруженно первое неравенство.

Student.java - клас, конструктор
StudentComparison.java - два методи, що порівнюють параметри студентів одним if і кількома вкладеними if
