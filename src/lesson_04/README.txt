номер студенческого билета, имя, фамилию, год обучения, средняя оценка по математике,
средняя оценка по экономике, средняя оценка по иностранному языку. 2-ой класс назовите StudentTest.
В нём Вы должны будете создать 3 разных объекта классаStudent, вывести на экран в читабельном
виде среднюю арифметическую оценку каждого студента (учитывая все 3 предмета).
