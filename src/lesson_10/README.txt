Урок 10. Понятия «import» и «import static». Использование комментариев (ст 132-137)
Import, import static

Создайте пакет p1, в нём класс А. В пакете p1 создайте подпакет p2, в нём класс B со static элементами.
В подпакете p2 создайте подпакет p3, в нём класс C. Создайте новый пакет p4, в нём класс D.
В пакете p4 создайте подпакет p5, в нём класс E. Внутри класса D напишите код, который будет задействовать
любые элементы классов A, C, E, а также static элементы класса B. Используя выражения import и import static,
работайте с не полными именами классов.

в класі package1.A є тільки клас з параметрами
в класі package1.package2.B клас з статичними параметрами і статичний метод
в класі package1.package2.package3.C клас з двома методами, в яких використала сканер консолі
в класі package4.package5.E клас з параметрами, конструктор і метод
в класі package4.D використовую параметри, створюю об'єкти і викликаю методи з інших класів
