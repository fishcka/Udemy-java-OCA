Урок 18 Понятие «массив». Работа с массивами. (ст. 217-226)
Arrays. class Arrays and its methods

1. Создайте класс, в котором создайте метод sortirovka. Инпут параметром данного метода будет одномерный массив типа
int. Метод должен возвращать уже отсортированный по возрастанию массив .
Продемонстрируйте данный метод.
2. Создайте класс, в котором создайте метод showArray. Инпут параметром данного метода будет двумерный массив типа
String. Метод должен выводить на экран данный массив в следующем виде:
{ {элемент00, элемент01}, {элемент10}, {элемент10, элемент11} }
Содержимое,естественно, будет зависеть от Вашего массива.
Продемонстрируйте данный метод.

SortArray.java: використала методи Arrays.sort(array), Arrays.toString
ShowArray.java: використала методи Arrays.deepToString, Arrays.replace
