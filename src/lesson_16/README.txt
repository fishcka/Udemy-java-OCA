Урок 16. Всё о классе String. Методы String (ст 194-200)

Создайте класс, в котором создайте метод email. Данный метод должен принимать в инпут 1 String параметр. Данный
параметр должен содержать в себе email-ы в следующем виде: ya@yahoo.com; on@mail.ru; ona@gmail.com; ,
т.е. после каждого e-mail должен стоять знак препинания “;”. Ваш метод должен выводить на экран информацию о том, какой
почтой пользуются, исходя из параметра, т.е. оутпут должен быть следующего вида:
yahoo
mail
gmail
Продемонстрируйте данный метод.
