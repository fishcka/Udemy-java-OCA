Урок 11. Использование примитивных и ссылочных типов данных при вызове метода (ст 143-145)
Using primitive and reference data types when calling a method

Создайте класс Car с тремя переменными: цвет, мотор и количество дверей. Затем создайте класс CarTest,
в котором должны быть 2 метода. 1-ый изменяет количество дверей объекта класса Car на количество,
прописанное в параметре метода. 2-ой принимает в параметры 2 объекта класса Car и меняет их цвета местами.
Примените оба метода в main и выведите на экран атрибуты всех объектов.
