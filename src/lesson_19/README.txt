Урок 19 Параметры метода типа varargs. Аргументы типа command line. Цикл "foreach" (ст. 232-238)
Command line arguments.

Создайте метод abc, инпут параметр которого – N-ое количество массивов типа String. В методе создайте новый
массив, который будет состоять из элементов массивов- параметров и будет оутпутом данного метода. В методе main
вызовите метод abc и его элементам, которые равны значениям command line параметров, присвойте значение
null. Выведите элементы обновлённого массива на экран.
Везде, где возможно, используйте foreach loop.
Запустите приложение с командной строки.

запускаем терминал и указываем путь к джаве
set path=C:\Program Files\Java\jdk-14\bin

переходим в директорию с файлом программы и компилируем
javac Programm.java

запускаем программу
java Programm

передаем параметры в командной строке
java Programm one two three four five six
